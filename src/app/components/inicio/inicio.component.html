Instrucciones de Instalación

  1. Descargar ambos repositorios.
  2. Tener Angular 16 instalado.
  3. Tener .NET Core versión 8 instalado.
  4. Instalar las dependencias del proyecto Frontend con npm install.
  5. Correr la aplicación Angular con npm start.
  6. Compilar la aplicación de .NET Core 8 (Mac o Windows): usar el comando dotnet build (ruta de la app).
  7. Correr la aplicación de .NET Core 8 (Mac o Windows): usar el comando dotnet run (ruta de la app).
  8. Nota: También puedes usar Visual Studio para correr la aplicación.
  9. Listo :)
